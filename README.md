# Superbowl-Score-Generator

This school project predicts the 2024 Super Bowl score using webscraping and APIs.

## Table of Contents
- [Project Overview](#overview)
- [Features](#features)
- [Usage](#usage)

## Project Overview
Superbowl Score Generator is a small school project that gathers the team records and statistics per game from the Chiefs and Eagles for the 2024 Superbowl. It can be improved by getting more years of data or more player-specific data.

## Features
- Web scraping: pulls data from web sources to gather the overall team record and final scores throughout the season
- API: uses public NFL data API to fetch more game specific data
- Score prediction: uses a linear regression model with hyperparameters to predict final score

## Usage
- python MH_API_SuperBowl_Score_Generator.py
